NAME

IPlant::TreeRec::BlastArgs

VERSION

This documentation refers to IPlant::TreeRec::BlastArgs version 0.0.2

SYNOPSIS

        use IPlant::TreeRec::BlastArgs

        # Gets the sequence.
        $sequence = $blast_args->get_sequence();

        # Creates a BLAST arguments object from a JSON string in the format:
        $blast_args = IPlant::TreeRec::BlastArgs->from_json($json);

        # Builds the command to execute in order to perform the BLAST search.
        @command = $blast_args->build_command( $exe_dir, $db_dir );

get_sequence

Usage

$sequence = $blast_args->get_sequence();

Purpose

Gets the sequence.

Returns

The sequence.

Parameters

None.

Throws

No exceptions.

_new

Returns

The new set of blast arguments.

Parameters

executable - the name of the executable file. sequence - the query sequence. database - the name of the database file. evalue - the evalue threshold. max_num_seqs - maximum number of sequences to return.

Throws

No exceptions.

Usage : $blast_args = IPlant::TreeRec::BlastArgs->_new( { executable => $executable_path, sequence => $sequence, database => $database_file_name, evalue => $evalue, max_num_seqs => $max_num_seqs, } ); Purpose Creates a new set of BLAST arguments.

from_json

Usage

$blast_args = IPlant::TreeRec::BlastArgs->from_json($json);

Purpose

Creates a BLAST arguments object from a JSON string in the format:

Parameters

$json - the JSON string.

Throws

No exceptions.

Available sequence types are "nucleotide" and "protein". The type of sequence will be guessed at when not provided. The evalue is float (ie. 0.00001) or sci notation (ie. 1E-5) that represents the maximum threshold e-value for reporting matches in the BLAST database. The evalue defaults to 0.01 if not defined. The maxNumSeqs is an integer representing the maximum number of sequences to return from the BLAST search. The default value of maxNumSeqs is 200. Returns The new BLAST arguments object.

build_command

Usage

@command = $blast_args->build_command( $exe_dir, $db_dir );

Purpose

Builds the command to execute in order to perform the BLAST search.

Returns

The command.

Parameters

$exe_dir - the path to the executable file directory. $db_dir - the path to the database file directory.

Throws

No exceptions.
