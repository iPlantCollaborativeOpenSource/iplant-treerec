NAME

IPlant::TreeRec::DatabaseTreeLoader

VERSION

This documentation refers to IPlant::TreeRec::DatabaseTreeLoader version 0.0.2

SYNOPSIS

        use IPlant::TreeRec::DatabaseTreeLoader

        # Initializes a new tree loader with the given database handle.
        $loader = IPlant::TreeRec::DatabaseTreeLoader->new($dbh);

        # Loads the species tree with the given tree name.
        $species_tree = $loader->load_species_tree($tree_name);

        # Loads the gene tree for the given gene family name.
        $gene_tree = $loader->load_gene_tree($family_name);

SUBROUTINES/METHODS

_count_edge_dups

Usage

$count = $loader->_count_edge_dups($node_id);

Purpose

Counts the number of duplication events on the edge leading into the node with the given ID.

Returns

The number of duplication events.

Parameters

$node_id - the ID of the node that the edge leads into.

Throws

No exceptions.

load_species_tree

Usage

$species_tree = $loader->load_species_tree($tree_name);

Purpose

Loads the species tree with the given tree name.

Returns

The species tree.

Parameters

$tree_name - the name of the tree to load.

_count_dups

Usage

$count = $loader->_count_dups( $node_id, $is_on_node );

Purpose

Counts the number of duplication events either on the node with the given ID or on the edge leading into that node. If $is_on_node is true then the duplication count for the node itself is returned. Otherwise, the duplication count for the edge leading into the node is returned.

Returns

The number of duplications.

Parameters

$node_id - the ID of the node to check. $is_on_node - true if we should count duplications on the node itself.

Throws

No exceptions.
