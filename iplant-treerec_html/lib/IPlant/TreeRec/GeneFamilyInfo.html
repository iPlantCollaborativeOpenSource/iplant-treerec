NAME

IPlant::TreeRec::GeneFamilyInfo

VERSION

This documentation refers to IPlant::TreeRec::GeneFamilyInfo version 0.0.2

SYNOPSIS

        use IPlant::TreeRec::GeneFamilyInfo

        # Creates and initializes a new object instance.
        $info = IPlant::TreeRec::GeneFamilyInfo->new( { dbh => $dbh, go_term_length_limit => $limit, go_categories => \@categories, } );

        # Obtains summary information for a gene family.
        $summary_ref = $info->get_summary( $family_name, $species_tree_name );

        # Obtains detail information for a gene family. Currently, the detail information is the same as the summary information except that the entire list of GO annotatins is included in the detail information.
        $details_ref = $info->get_details( $family_name, $species_tree_name );

SUBROUTINES/METHODS

_get_all_go_terms

Usage

@go_terms = $info->_get_all_go_terms($protein_tree_id);

Purpose

Retrieves the GO terms for the given protein tree identifier.

Returns

The list of GO terms.

Parameters

$protein_tree_id - the protein tree identifier.

Throws

No exceptions.

_count_genes

Usage

Purpose

Counts the number of genes in the given gene family ID.

Returns

The number of genes.

Parameters

$family_id - the identifier of the gene family to examine.

Throws

No exceptions.

_get_representative_go_term

Usage

$term = $info->_get_representative_go_term( $protein_tree_id );

Purpose

Parameters

Throws
